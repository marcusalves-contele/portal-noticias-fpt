# Design System - Contele Fleet

## Cores

| Token | Hex | Uso |
|-------|-----|-----|
| Purple (primario) | `#8B23E5` | Logo, CTAs, destaques |
| Purple Deep | `#4C1D95` | Hover, sombras |
| Charcoal | `#20252E` | Texto, wordmark no logo |
| Dark BG | `#18122B` | Fundos escuros (verso de materiais) |
| Cream | `#FAFAF8` | Fundo claro |
| White | `#FFFFFF` | Cards, texto em fundo escuro |

**CMYK-friendly**: O roxo `#8B23E5` imprime bem. Evitar roxos neon como `#7C3AED` que podem não reproduzir em CMYK.

## Impressao

- **Minimo**: 8px para qualquer texto impresso
- **Sangria**: 3mm em todos os lados
- **Safe zone**: 5mm de margem interna
- **Formato padrao**: A5 landscape (210x148mm) para cartoes
